# Images

Figures, diagrams, logos reused across lectures and notebooks.

## Format hints

- **SVG** where possible. Scales to any size without blur. Ideal for diagrams,
  logos, plots exported from code.
- **PNG** for screenshots and pixel art (UI captures, terminal screenshots).
- **JPG** *only* for photographs. Don't use it for diagrams — JPEG compression
  creates ugly artefacts around crisp edges and text.

## Naming

- Lowercase, hyphen-separated: `course-overview.svg`, `pipeline-step-1.png`.
- ASCII only (no *æ, ø, å*) in filenames.
- Give files context in the name, not just `image-1.png` — future-you will
  thank present-you.

## Hygiene

If an image hasn't been referenced in two semesters, it probably doesn't need
to be here. Images accumulate silently; prune them during the January reset.
