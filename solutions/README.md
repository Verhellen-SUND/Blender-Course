# Solutions

**Instructor-facing answers.** Students should not see what's in here until
after the relevant deadline.

## Three common patterns

Pick one of these — they all work, the right choice depends on your course:

### 1. Keep solutions in a **separate private repository**

The least risky option. Link to it from your syllabus, not from this public
repo's README. Students never see a link.

### 2. Keep solutions here, but release via **GitHub Releases**

Add solutions to this folder locally, but add `solutions/` to `.gitignore`
for the public branch. When the deadline passes:

1. Copy the relevant solution into a tagged release's attachments.
2. Or flip the folder to public with a single commit.

### 3. Send solutions through the institution's **LMS** only

Keep GitHub purely for the public-facing material. Best for sensitive
answer keys (e.g. exam-style questions).
